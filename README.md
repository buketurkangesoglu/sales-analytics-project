# Sales Analytics Project

Bu proje, satış verilerini analiz etmek için hazırlanmış kapsamlı bir Python projesidir.  
Proje boyunca SQL veri tabanları oluşturuldu, Python ile veri analizi yapıldı ve sonuçlar görselleştirildi. Bunun haricinde bu tablolar Power BI da dashboard oluşturuldu 

## Klasör Yapısı

- `data/` → Veri dosyaları (CSV veya Excel formatında)  
- `scripts/` → Python analiz ve görselleştirme dosyaları  
- `images/` → Analizlerden elde edilen görseller  
- `reports/` → Hazırlanan raporlar ve çıktı dosyaları  

## Proje Adımları

1. **SQL Tablo Oluşturma**  
   - Müşteri, ürün ve satış tabloları oluşturuldu.  
   - Veri bütünlüğü için primary ve foreign keyler tanımlandı.  

2. **Python ile Veri Analizi**  
   - Pandas, NumPy ve Matplotlib kullanılarak veriler işlendi ve görselleştirildi.  
   - Farklı analizler ve raporlar oluşturuldu.  

3. **Git ve GitHub ile Versiyon Kontrolü**  
   - PyCharm kullanılarak proje Git ile takip edildi.  
   - İlk commit ve push işlemleri yapıldı.  
   - `.gitignore` ile gereksiz dosyalar yoksayıldı.
